# InterviewQuestions.md 格式规范

## InterviewQuestions.md 格式规范

文件开头包含简历摘要，> **简历摘要**：{学校} / {学历} / {专业} / {期望薪资} / {技能列表} / {项目经历-项目名称}

然后按维度组织问题，每个问题预留答题区域：

### 一、学历与学校

**Q1**：{问题原文}

> 回答：___（待用户填写）___

**Q2**：{问题原文}
> 回答：___（待用户填写）___

### 二、专业能力 — {技能1}

**Q3**：{问题原文}
> 回答：___（待用户填写）___

...（依此类推）

格式约束：
- 问题前不加序号前缀干扰排版（直接用 **Q{n}**）
- 每个回答区用 > 引用块包裹，填充提示统一为"___（待用户填写）___"
- 维度间用 ### 三级标题分隔
- 从 WrQuestions.md 抽取代入的历史问题，格式为 **Q{n}**（回顾）：{问题原文}，回答区不变

> 若本文件不存在或无法读取，SKILL.md 中已备默认格式作为兜底。
